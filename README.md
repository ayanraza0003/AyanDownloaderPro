# ✨ Features

* 🎥 Download YouTube videos in the highest available quality
* 🎵 Download audio as MP3
* 📱 Download supported Instagram media
* 🖼️ Thumbnail preview before downloading
* 📊 Live download progress
* 📂 Choose download location
* ⚡ Automatic FFmpeg integration
* 💜 Modern CustomTkinter interface
* 📦 Windows installer included

# 🚀 Installation

## Option 1 (Recommended)

1. Download **AyanDownloaderPro_Setup_v1.0.0.exe**
2. Run the installer.
3. Launch AyanDownloaderPro from the Desktop or Start Menu.

---

## Option 2 (Run from Source)

```bash
git clone https://github.com/ayanraza0003/AyanDownloaderPro.git
cd AyanDownloaderPro

pip install -r requirements.txt

python main.py
```
