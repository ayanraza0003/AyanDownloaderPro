## [v1.0.0] - 2026-06-27

### 🎉 Initial Public Release

### Added

* Modern desktop UI built with CustomTkinter
* YouTube video downloading
* MP3 audio extraction
* Instagram media downloading
* Video thumbnail preview
* Download progress tracking
* Automatic FFmpeg integration
* Windows installer
* GitHub Releases support

### Improved

* Better user interface
* Cleaner project structure
* Bundled dependencies for standalone distribution

### Fixed

* FFmpeg packaging
* Asset loading after PyInstaller build
* Installer generation issues
* Thumbnail preview restoration
